# Little Journey

Baby milestone journal app for Singapore market (Expo, iOS + Android).

## Commands

```bash
npm run lint    # ESLint flat config
```

## Key Config

- Expo SDK 54, managed workflow
- New Architecture + React Compiler enabled
- Strict TypeScript, typed routes
